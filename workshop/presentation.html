Hva gjør vi i dag?

  • Hacke en Minecraft-server ved å modde klienten
  • Finn flagg gjemt rundt på serveren
  • Post dem på CTFd-serveren

Litt som Burp Suite for HTTP — i stedet for å intercepte nettverkspakker, injiserer vi kode i klienten og endrer oppførselen fra innsiden.

Hvorfor Minecraft?

  • Komplekst system med veldefinerte grensesnitt
  • Klient–server-arkitektur lik moderne applikasjoner
  • Visuelt og gøy — du ser hva som skjer
  • Samme tankesett som sikkerhetstesting av ekte systemer

Hvordan?

  • Vi skriver en Fabric mod — et Java-program som kjører inne i Minecraft-klienten
  • Moden kjører lokalt hos deg. Serveren vet ikke om den.
  • Vi bruker hooks (events) til å koble oss på hendelser i spillet

onInitializeClient()        ← kjøres én gang ved oppstart
    └── event.register()    ← "når X skjer, kjør dette"
            └── din kode    ← kjøres når X faktisk skjer

Demo — Oppsett

  1. Klon repoet — husk tag 1.21.11! 
    git clone --branch 1.21.11 \
  https://github.com/FabricMC/fabric-example-mod
  2. Åpne i IntelliJ IDEA og la Gradle importere ferdig
  3. Generer Minecraft-kilder: 
    Gradle → Tasks → fabric → genSources
  4. Velg Minecraft Client i Run-menyen og trykk Run

Etter genSources kan du Ctrl+Klikke deg inn i hvilken som helst Minecraft-klasse.

Demo — Første mod

Vis en melding i chatten når spilleren joiner en verden.

ClientPlayConnectionEvents.JOIN.register((handler, sender, client) -> {
    client.player.displayClientMessage(
        Component.literal("Velkommen til MC Hacking! 🎉"),
        false
    );
});

Funnet ved å søke i Fabric API etter join-events → ClientPlayConnectionEvents → lese Javadoc på JOIN

Demo — Keybinding

Trykk en knapp → send en chatmelding.

Registrer knappen én gang:

var key = KeyBindingHelper
  .registerKeyBinding(
    new KeyMapping(
      "key.examplemod.action",
      KEYSYM,
      GLFW.GLFW_KEY_R,
      category
  ));

Poll hvert tick:

ClientTickEvents
  .END_CLIENT_TICK
  .register(client -> {
    while (key.consumeClick()) {
      // din kode her
    }
  });

→ docs.fabricmc.net/develop/key-mappings

Hvordan finne riktig hook

  1. Beskriv hva du vil: "gjør noe når spilleren hopper"
  2. Søk i Fabric docs eller bla i fabric-api kildekoden
  3. Les Javadoc på eventet — sjekk callback-signaturen
  4. Ingen event? Bruk en Mixin for å injisere kode direkte i en Minecraft-metode

→ Mixin intro · SpongePowered Mixin wiki

Koble til serveren

  1. Verifiser at du er medlem av appsec-ctf namespacet i console.nais.io
  2. Start port-forward: 
    kubectl port-forward -n appsec-ctf \
  $(kubectl get pod -n appsec-ctf -l app=minecraft-ctf \
    -o jsonpath='{.items[0].metadata.name}') 25565:25565
  3. Åpne Minecraft → Multiplayer → Direct Connection
  4. Server address: 127.0.0.1
